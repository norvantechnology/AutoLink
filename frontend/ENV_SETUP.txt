🔄 SWITCHING BACKENDS
────────────────────────────────────────────────────────────

TO USE LOCALHOST BACKEND:
1. Stop dev server (Ctrl+C)
2. Rename files:
   mv .env.local .env.local.ec2
   mv .env.local.localhost .env.local
3. Restart: npm run dev

TO USE EC2 BACKEND (DEFAULT):
1. Stop dev server (Ctrl+C)  
2. Rename files:
   mv .env.local .env.local.localhost
   mv .env.local.ec2 .env.local
3. Restart: npm run dev
